## Home Challenge

## Prerequisites 🧰
- [x] Node >= 18

## Quick Start 🏁

Install the dependencies:

```sh
npm install
```

Start on iPhone simulator:

```sh
npm run ios
```

Start project using expo:

```sh
npm run start
```
Then, press `a` to open Android  or `i` to open on iOS simulator 

## Running the lint:

```sh
npm lint
```
and run this command to fix the code
```sh
npm lint:fix
```
